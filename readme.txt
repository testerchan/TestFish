「TestFish」をダウンロードしてくださりありがとうございます。
このテストツールは、指定したドメイン内をランダムテスト的に勝手に動き回り、ページ内のリンクのhttp status確認（400や500など）や、ブラウザのコンソールに表示されるエラーの確認、ページ内のボタン、リンク、ラジオボタン、チェックボックスなどをクリックして動作を確認するツールです。
つまり、ドメイン内のページを勝手に回遊してボタンなどをポチポチ押しまくってエラーが出てないか確認してくれるツールです。
ログはlogフォルダにたまるようにしています。



■使用環境
・windowsのみ
・chrome74（他のverのchromeの場合はchromedriver.exeを変更する必要があります。http://chromedriver.chromium.org/downloads）
・seleniumなどは特にインストールする必要はありません。main.exeをクリックすると動きます。

■使用方法
settings.txtを設定し、使用してください。

■settingsの項目の意味
TargetUrl……TestFishが回遊するドメイン。このドメイン以外につながるリンクなどはチェックしません。
ActuatingCycle……この回数分だけTestFishが行動を行います。
WaitTimeForAction……次の行動に移るまでの待ち時間です。初期値は5秒です。整数値で指定してください。
WaitTimeForLinkCheck……リンクをチェックするときの待ち時間です。サーバー負荷軽減のために指定します。自分のサイトの場合は0にしても良いと思います。
LogDir……ログを保存するフォルダです。
IsHeadless……TrueにするとchromeをHeadlessモード（画面が見えないモード）で動かします。全画面キャプチャが欲しい場合はこのモードにする必要があります。Falseの場合はブラウザが起動して起動している様子が見えます。
IsLinkCheck……Trueの場合、リンクのステータスチェックを行います。Falseの場合はチェックを行いません。
IsBrowserCheck……Trueの場合、ブラウザに表示されるコンソールを取得します。Falseの場合は取得を行いません。
